# scheduler-sat

*Work in progress*

> Using a SAT solver to generate and prove correctness of schedules given a list of constraints

## Quick Start

Requirements:

- Python 3

Clone and run tests:

```sh
$ git clone git@github.com:noahbass/scheduler-sat.git
$ cd scheduler-sat
$ python3 main.test.py -v
```
